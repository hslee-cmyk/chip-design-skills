# .ai/design-knowledge/ — RTL 설계 지식

RTL 설계에 직접 영향을 주는 도메인 지식을 모은다. graphify가 색인한다.

- `hardip-*.md` — FPGA 하드 IP 사용법·제약 (SB_IO, SPRAM, PLL 등)
- `known-issues.md` — 툴/실리콘 known issue + 회피책
- `*-race.md`, `*-bug.md` — 재현된 RTL/타이밍 버그 분석

> 운영·인프라 가이드(build, servers, MCP 등)는 `.ai/ops/` 에 둔다.
> 해결된 버그 솔루션은 `docs/solutions/<category>/` 에 자산화한다 (`/solution-capture`).
